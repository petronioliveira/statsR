# statsR
Repositório educacional dirigido à Bioestatística usando o R
Este livro foi concebido como uma introdução à bioestatística usando a linguagem R. 
O objetivo é que ele seja um instrumento para facilitar e entender as análises estatísticas, principalmente, para aqueles que estão se iniciando nesta área. 
Esperamos que ele facilite a assimilação dos conteúdos aqui colocados
